# AutoParts Platform 🚗📦

**Повноцінна ERP/E-commerce платформа для управління магазином автозапчастин з глибинною інтеграцією даних та оптимізацією складських процесів.**

---

## 🎯 Про проєкт
Це не просто інтернет-магазин, а спроба побудувати гнучку архітектуру для складних технічних даних. Проєкт розробляється з урахуванням специфіки підбору запчастин, крос-номерів та реальних складських залишків.

### 💻 Технологічний стек
* **Backend:** [Laravel 12](https://laravel.com/) (найсвіжіша версія)
* **Admin Panel:** [Filament v5](https://filamentphp.com/) (потужний інтерфейс управління)
* **Database:** MySQL 8+ (використання UUID, JSON-полів та складних індексів)
* **Architecture:** Service Layer pattern, Nested Sets для категорій.

---

## 🛠 Що вже реалізовано (Status: Active Development)

### 🏗 Ядро та Товари
- [x] **Advanced Product Engine:** Підтримка УКТ ЗЕД, UUID, складних характеристик та специфікацій.
- [x] **Smart Categories:** Деревоподібна структура (Nested Sets) з автоматичним перерахунком кількості товарів у гілках.
- [x] **Category Mirrors:** Система "дзеркальних" категорій для зручної навігації.

### 📦 Склад та Логістика (Stock Management)
- [x] **Multi-Location Support:** Облік залишків у розрізі складів та конкретних локацій (ряд/полиця).
- [x] **Reservation System:** Механізм резервування товарів для запобігання помилкам при замовленні.
- [x] **Performance Optimization:** Окремі таблиці `StockSummary` та спеціалізовані індекси для миттєвого пошуку.
- [x] **Delivery Logic:** Розрахунок термінів доставки (min/max) від складів до точок видачі.

### 🎨 Контент та SEO
- [x] **Main Page Builder:** Керування блоками товарів та групами на головній сторінці прямо з адмінки.
- [x] **SEO Module:** Налаштування мета-даних, кастомних підписів (captions) для категорій та магазинів.
- [x] **Multilingual:** Повна підтримка локалізації (UA/EN).

## 📦 Встановлення (Development)

1. **Склонуйте репозиторій:**
   ```bash
   git clone [https://github.com/lykach/AutoParts.git](https://github.com/lykach/AutoParts.git)
   cd AutoParts
Встановіть залежності:

Bash
composer install
npm install && npm run build
Налаштуйте окруження та запустіть міграції:

Bash
cp .env.example .env
php artisan key:generate
php artisan migrate --seed
Оновіть дерево категорій:

Bash
php artisan categories:rebuild
