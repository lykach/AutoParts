AutoParts Platform 🚗📦
Повноцінна ERP/E-commerce платформа для управління магазином автозапчастин з глибинною інтеграцією даних та оптимізацією складських процесів.

🎯 Про проєкт
Це не просто інтернет-магазин, а високонавантажена система для роботи з технічними даними запчастин. Проєкт побудований на принципах гнучкої архітектури, що дозволяє обробляти мільйони артикулів, крос-номерів та залишків у реальному часі.

💻 Технологічний стек:
Backend: Laravel 12 (Core Engine)

Admin Panel: Filament v3+ (Advanced Admin Interface)

Database: MySQL 8+ (з використанням складних індексів та JSON-полів)

Architecture: Service Layer pattern, UUID identification, Nested Sets для категорій.

🛠 Що вже реалізовано (Status: Active Development):
🏗 Ядро та Товари
[x] Advanced Product Engine: Підтримка УКТ ЗЕД, UUID, складних характеристик.

[x] Smart Categories: Деревоподібна структура (Nested Sets) з автоматичним перерахунком кількості товарів у гілках.

[x] Category Mirrors: Система "дзеркальних" категорій для гнучкої навігації.

📦 Склад та Логістика (Stock Management)
[x] Multi-Location Support: Облік залишків у розрізі складів та конкретних локацій (ряд/полиця).

[x] Reservation System: Механізм резервування товарів для запобігання оверсейлу.

[x] Performance Optimization: Окремі таблиці StockSummary та спеціалізовані індекси для миттєвого відображення наявності.

[x] Delivery Logic: Розрахунок термінів доставки (min/max) від складів до магазинів.

🎨 Контент та SEO
[x] Main Page Builder: Керування блоками товарів та групами на головній сторінці прямо з адмінки.

[x] SEO Module: Налаштування мета-даних, кастомних підписів (captions) та SEO-полів для магазинів та категорій.

[x] Multilingual: Повна підтримка локалізації (UA/EN).

🚀 Найближчі плани:
TecDoc Integration: Синхронізація з каталогами виробників та специфікаціями.

Cross-Number Engine: Інтелектуальний пошук аналогів та замінників.

API Layer: Створення REST API для фронтенду та мобільних додатків.

📦 Встановлення (Development)
Склонуйте репозиторій:

Bash
git clone https://github.com/lykach/AutoParts.git
Встановіть залежності:

Bash
composer install
npm install && npm run build
Налаштуйте окруження та запустіть міграції:

Bash
cp .env.example .env
php artisan key:generate
php artisan migrate --seed
Оновіть дерево категорій (після міграції):

Bash
php artisan categories:rebuild
